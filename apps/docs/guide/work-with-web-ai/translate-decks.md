# Translate Decks

Translation can target selected text, the current page, or the full deck.

## Why It Matters

Manually translating a presentation is slow and error-prone. LocalStudio keeps translation inside the deck so text updates in place.

## Setup

- Choose a translation model.
- Choose the target language.
- Prepare the required browser or WebGPU provider.
