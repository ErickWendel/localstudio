# Prompt to Slide

Prompt to Slide turns a plain-language request into editable slide objects.

## Result

The output should be text, image, and shape layers that stay editable instead of becoming a flat bitmap.

## Requirements

- A compatible prompt provider.
- Browser/model setup from AI Tools.
- Enough local storage for browser-managed model caches when using external WebGPU models.
