# GIPHY

GIPHY provides GIF search inside the Elements panel.

## Configure

1. Open media integration settings.
2. Add a GIPHY API key.
3. Return to Elements and search GIFs.
4. Insert a result as a normal GIF layer.

## Usage

Inserted GIFs become editable media layers. They can be arranged, animated, and toggled for playback.
