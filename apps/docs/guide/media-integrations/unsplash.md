# Unsplash

Unsplash provides stock image search inside the Elements panel.

## Configure

1. Open media integration settings.
2. Add an Unsplash access key.
3. Return to Elements and search images.
4. Insert a result as a normal image layer.

## Usage

Inserted images can be cropped, flipped, arranged, animated, and used as source material for other editing workflows.
