# Presenter Mode

Presenter Mode moves the deck from editing to delivery.

## Presentation Options

- Play presentation.
- Present in fullscreen.
- Open Presenter view.
- Play from beginning.

## Presenter Remote

The remote-control flow uses session-scoped peer transport to reduce latency between presenter controls and the deck.
