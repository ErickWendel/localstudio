# Sharing

Sharing creates public-facing payloads from a mirrored project.

## How It Fits

The editable source of truth stays local. The public viewer needs reachable assets from mirrored storage.

## Share Surface

- Open the editor Share action.
- Create or copy the public link.
- Use the link or embed where supported.
