# PowerPoint (.pptx)

Use PowerPoint import when an existing deck should become a LocalStudio project.

## Workflow

1. Export from Google Slides, Keynote, or PowerPoint as `.pptx`.
2. In LocalStudio, open File > Import > PowerPoint (.pptx).
3. Review the imported slides as editable objects.
4. Save to a local project folder when you want local history and mirroring.

## Result

Imported content becomes a LocalStudio document where supported text, image, shape, and media layers can keep moving through the editor workflow.
