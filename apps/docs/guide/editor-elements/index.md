# Editor Elements

LocalStudio projects are built from editable slide objects. Imports and AI actions should return to these objects instead of flattening the deck into a screenshot.

## Element Types

- [Text](/guide/editor-elements/text): content, typography, opacity, arrange, and typing animations.
- [Images](/guide/editor-elements/images): imports, generated images, crop, flip, background removal, and image animation behavior.
- [Videos and GIFs](/guide/editor-elements/videos-and-gifs): playback, replacement, movie start, and presenter playback.
- [Shapes](/guide/editor-elements/shapes): shape insertion, fill, border, line endpoints, arrange, and line-draw animation.

## Shared Controls

Most elements share opacity, layer order, alignment, lock state, and animation build controls. The detailed pages document those controls where users encounter them.
