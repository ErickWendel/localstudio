# Images

Image layers can come from imports, clipboard, generated image workflows, stock search, or image editing results.

## Add Images

- Import images from disk.
- Insert Unsplash results from Media Integrations.
- Generate images from Web AI when the image model is ready.

## Edit Images

- Crop an image layer.
- Flip an image.
- Remove a background with the AI image editing workflow.
- Adjust opacity and arrange position like other elements.

## Animate Images

Images support normal element builds such as reveal, wipe, move, and directional effects.
