# Slide Layouts

Slide layouts define reusable placement structure for titles, body content, footers, and slide numbers.

## Supported Controls

- Choose a slide layout.
- Apply a layout to the active slide.
- Toggle placeholder visibility for title, body, footer, and slide number.
- Set slide background color.

LocalStudio does not currently document themes as a supported user workflow.
