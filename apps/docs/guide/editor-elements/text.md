# Text

Text layers are normal editable objects. You can insert preset text, edit the content, change typography, arrange the layer, and animate it.

## Add And Edit Text

- Use the Text panel to add a heading, subheading, or body text.
- Use the selected element inspector to edit the text content.
- Use the Design controls to adjust font family and other text styling.

## Fonts

LocalStudio supports bundled font choices and local/imported fonts where the browser grants access. Fonts can also be mirrored for public viewers when project mirroring is configured.

## Arrange And Animate

- Use arrange controls to move text forward, backward, front, or back.
- Use alignment controls to center text horizontally, vertically, or on the page.
- Text supports general element builds and text-specific keyboard typing animation.
