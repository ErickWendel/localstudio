# Shapes

Shapes provide structure for diagrams, callouts, dividers, and slide composition.

## Add Shapes

The Elements panel includes circles, lines, rectangles, rounded rectangles, triangles, pentagons, diamonds, parallelograms, arrows, and arcs.

## Style Shapes

- Set fill and border.
- Configure line endpoints where supported.
- Adjust opacity.
- Arrange with front/back/forward/backward controls.

## Animate Shapes

Shapes support general element builds and shape-specific line draw behavior.
