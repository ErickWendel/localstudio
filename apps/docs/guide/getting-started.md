# Getting Started

LocalStudio.dev is a browser-native slide editor for people who want editable decks without handing the whole workflow to a remote workspace.

Cloud presentation tools are convenient until sync becomes the product risk. The reason LocalStudio exists is practical: iCloud sync issues can lose or corrupt entire presentations, and translating a deck manually is still slow. LocalStudio keeps the deck as editable local layers and can translate the whole deck with one action when the browser AI setup is ready.

## Start

1. Open LocalStudio in Chrome.
2. Create a project or import an existing deck.
3. Save the project to a local folder when you want version history, mirroring, and safer long-running workflows.
4. Use Web AI tools when your browser and local model setup are ready.

## Recommended Surface

- Chrome for Chrome-first browser AI and File System Access APIs.
- At least 10GB free storage for model caches, generated assets, and local history.
- A project folder you control for local project persistence.
