# FAQ

## Is LocalStudio a cloud editor?

No. The editor runs in the browser and project files can stay in a folder you control.

## Does LocalStudio support themes?

Themes are not documented as a supported user workflow right now.

## Does hosting LocalStudio host my AI models?

No. Browser AI model caches and Chrome built-in AI setup remain on the client.

## Where do I report bugs?

Use the [GitHub issue chooser](https://github.com/ErickWendel/localstudio/issues/new/choose).
